Dit bestand laat drie mogelijke UI-varianten zien voor de sectie "User stories of taken". Kies er één (of een combinatie) om in de app door te voeren. Items zijn niet afvinkbaar; sprintplanning gebeurt op het werkbord.

Elk item toont een klein label (Uitgebreide user story / Eenvoudige userstory) zodat je in één oogopslag ziet wat het is.
